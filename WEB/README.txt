Initilisation du site:
	1. CREER LA BASE DE DONNE AVEC LE FICHIER: CREATDB.SQL
	2. CHANGER LES IDENTIFIANTS QUI VA BIEN POUR LA CONNEXION A VOTRE BASE DE DONNES DANS LE FICHIER CONNEXION/CONNECT.PHP
	3. ACTIVE VOTRE SERVER WEB
	4. OUVRIR UN NAVIGATEUR WEB
	5. localhost/Correct (attention: le dossier corrector dois trouver dans le dossier www)
	6. login dans la section admin
	7. puis inserer selon votre besoin les donnees dans votre base.
